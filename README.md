# Stickify

Stickify is a Next.js project initialized with vinext, so you can develop with either the standard Next.js workflow or the Vite-powered vinext workflow.

## Getting Started

Run one of the development servers:

```bash
npm run dev
npm run dev:vinext
```

Open [http://localhost:3000](http://localhost:3000) for Next.js or [http://localhost:3001](http://localhost:3001) for vinext.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
